## 📖 What is personal-page?

personal-page is a simple app that lets you turn your profile information into manga-style pages. These pages work well for GitHub READMEs or other online profiles. It uses AI and image generation tools to create multi-panel manga layouts that showcase your data in a clear, visual way.

You do not need programming skills to use it. The app runs in your browser or on your Windows PC with minimal setup.

## 🖥️ Using personal-page

### How to create manga pages

1. **Enter your profile data:** Use the form provided in the app to add details like your name, bio, skills, and social links.  
2. **Choose a template:** Pick from several manga panel styles that arrange your data visually.  
3. **Generate manga pages:** The app uses AI to create images and place panels automatically.  
4. **Preview your page:** Look over your manga-style profile.  
5. **Export files:** Save the cards or images directly or export markdown code for your GitHub README.

### Tips for best results

- Provide complete profile info for richer pages  
- Experiment with different templates to find your style  
- If you want, manually edit images or texts before export  
- Use the preview feature to check changes live

## 📚 Learn More

The app uses AI models related to image generation and diffusion, common in generative AI fields. It combines these with manga styling and profile layout design. The technologies involved include:

- Python scripting for backend tasks  
- Stable Diffusion for image generation

- Tailwind CSS for styling

All components work together to make manga-style profiles possible without complex manual design.
